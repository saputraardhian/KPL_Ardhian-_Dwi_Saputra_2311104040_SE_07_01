 1. `funtion.js`
File ini berisi contoh penggunaan berbagai bentuk fungsi di JavaScript:
- **Function Declaration**: `perkalian(a, b)`
- **Arrow Function**: `penjumlahan(a, b)`
- **Function Expression**: `pembagian(a, b)`

Output akan menampilkan hasil dari operasi aritmatika dasar antara dua angka.

 2. `if-else-switch.js`
File ini berisi contoh:
- Penggunaan struktur **`if-else if-else`** untuk memeriksa apakah suatu bilangan habis dibagi 2.
- Penggunaan **`switch-case`** untuk mengevaluasi nilai variabel `pilihan`.

Tujuannya adalah memahami pengambilan keputusan (decision making) dalam JavaScript.

 3. `variable-local-global.js`
File ini menunjukkan perbedaan antara:
- **Variabel Global**: `globaVariable`
- **Variabel Lokal**: `localVariable` di dalam fungsi `getMyIdentity()`
